## 📊 DBMS Lab: Database Creation & Implementation

During our recent DBMS lab session, we transitioned from theory to hands-on implementation. The goal was to build a functional local server environment to manage data effectively.

### 💻 Technical Execution
Building the database required a systematic approach to ensure data integrity:

*   **Environment:** Created a database using **MySQL** integrated with **WampServer**.
*   **Data Handling:** Developed a dataset with multiple features (columns) to simulate real-world attributes.
*   **Query Operations:** Successfully wrote and executed `INSERT` statements to populate the table with 3–5 rows of data.

> "This hands-on experience was crucial for understanding how data is structured and managed in real-world scenarios."
